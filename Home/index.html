Home

Welcome to Bimey's-Bit-Games! Here you can find some really cool games I've made. These games are made with Javascript, HTML, CSS, and Python. For the python games I've used the Turtle and Pygame librarys.

The minigames section has some basic games like Tic-Tac-Toe and Rock-Paper-Scissors. The programs section is just some cool programs that I've made. The best games section has my best games. The ones there are mostly made with Python. The other games section has games that don't fit into any other catagories.
